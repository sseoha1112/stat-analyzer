# 📊 StatAnalyzer — 통계 분석기

> 브라우저에서 바로 실행되는 심리학·사회과학 통계 분석 도구  
> **설치 없음 · R/SPSS 불필요 · 완전 무료**

🔗 **[라이브 데모 바로가기](https://sseoha1112.github.io/stat-analyzer/)**

---

## ✨ 주요 기능

### 통계 검정 (8종)
| 검정 | 용도 |
|------|------|
| 독립표본 t-검정 | 두 집단 평균 비교 |
| 일원 ANOVA | 3개 이상 집단 비교 |
| 이원 ANOVA | 두 독립변수 & 상호작용 |
| Mann-Whitney U | 비모수 두 집단 비교 |
| Kruskal-Wallis | 비모수 다집단 비교 |
| 카이제곱 (χ²) | 범주형 변수 독립성 검정 |
| 피어슨 상관분석 | 두 변수 간 선형 관계 |
| 단순 회귀분석 | 예측 및 관계 추정 |

### 그 외 기능
- **검정 자동 추천** — 변수 유형과 조건 수에 따라 적합한 검정 제안
- **정규성 검정** — Shapiro-Wilk (선택)
- **사후검정** — Tukey HSD (ANOVA 선택)
- **효과 크기** — Cohen's d, η², r 자동 계산
- **R 코드 자동 생성** — RStudio에 바로 복붙 가능
- **SPSS 코드 자동 생성** — SPSS Syntax 창에 바로 복붙 가능
- **AI 해석** — GROQ API 연동 시 결과 자동 해석 (한국어)

## 🚀 사용 방법

### Step 1 — 변수 설정
- **종속변수(DV)** 추가: 변수명 입력, 척도 유형 선택 (연속형 / 순위형 / 범주형)
- **독립변수(IV)** 추가: 변수명 입력, 조건명 입력 (예: `강의식`, `토론식`)
- 연구 목적 / 가설 입력 (선택, AI 해석에 활용)

### Step 2 — 데이터 입력
**엑셀 업로드 방식**
- `.xlsx`, `.xls`, `.csv` 파일 드래그&드롭 또는 클릭 업로드
- 열 매핑: DV와 IV에 해당하는 엑셀 열을 선택
- 엑셀의 집단 구분 값이 IV 조건명과 **정확히 일치**해야 함

**샘플 데이터 생성 방식**
- 조건당 N, 효과 크기(소/중/대) 설정 후 자동 생성
- 기능 테스트 용도

### Step 3 — 검정 실행
- 추천 검정 확인 후 원하는 검정 선택
- 유의수준 α 설정 (0.05 / 0.01 / 0.001)
- 사후검정, 효과 크기, 정규성 검정 옵션 선택
- **▶ 분석 실행** 클릭

## 📋 결과 화면 구성

```
기술통계 카드 (평균, SD, N)
└─ 각 조건별 요약 통계

정규성 검정 결과 (선택 시)
└─ Shapiro-Wilk W, p값

검정 결과
└─ 검정통계량, 자유도, p값, 효과 크기, 유의성

R 코드          ← RStudio에 복붙
SPSS 코드       ← SPSS Syntax에 복붙

AI 해석 (GROQ 연결 시)
└─ 결과 의미, 효과 크기 해석, APA 보고 형식
```

## 🤖 AI 해석 설정 (선택)

GROQ API를 연결하면 분석 결과를 한국어로 자동 해석해줍니다.

1. [console.groq.com](https://console.groq.com/keys) 에서 무료 API 키 발급
2. 우측 상단 **⚙ 설정** 클릭
3. API 키 입력 후 저장
4. 분석 실행 시 자동으로 AI 해석 생성

> 사용 가능한 모델: `llama-3.3-70b-versatile` (추천), `llama-3.1-8b-instant`, `mixtral-8x7b-32768`

## ⚠️ 정확도 안내

JS 통계 엔진은 대부분의 경우 R/SPSS와 거의 동일한 결과를 냅니다. 단, 아래 항목은 근사값입니다.

- **Shapiro-Wilk**: Royston 근사법 사용 (n ≤ 50 권장)
- **Tukey HSD**: q 임계값 고정값 사용

> 논문·학위논문·공식 보고서에는 R 또는 SPSS 코드로 재검증 후 사용을 권장합니다.
